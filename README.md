## 📋 功能简介

这是 ssngIRC运兵车 的用户端模组，功能：

- 自动构建拉人请求并发送到 SSNGIRC
- 告别繁琐的手动输入过程

## 📝 使用说明

1. 确保 irc运兵车 正在运行
2. 加载模组后 按键盘上“ ] } ”键打开配置ui，按“ \ | ”键发送快捷消息
3. 如果按键没反应，可以去**mc原版按键设置**自行修改
4. **使用前确保你的irc prefix不是 “.i” ！** 在Client里的ClientSettings里修改。否则你的请求会同时发到irc公共聊天里面 ！！！

### 前置要求

- Minecraft 1.21.4
- Fabric Loader
- Fabric API
- SouthSide客户端

## ⚙️ 配置

配置文件位于：`.minecraft/config/heypassed-carrier.json`

首次运行时会自动生成默认配置文件。

## 🛠️ 开发构建

```bash
# 克隆仓库
git clone https://github.com/yourusername/heypassed-carrier-user.git
cd heypassed-carrier-user

# 构建模组
./gradlew build

# 构建产物位于
build/libs/
```
